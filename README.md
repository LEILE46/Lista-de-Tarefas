# Lista-de-Tarefas
Objetivo: Criar e gerenciar tarefas simples diretamente no navegador.

Tecnologias: 
  HTML5 → estrutura da página  
  CSS3 → estilo visual  
  JavaScript puro (Vanilla JS) → lógica de interação  

Funcionalidades básicas: 

    Adicionar novas tarefas. 

    Exibir as tarefas em uma lista. 

    Remover tarefas ao clicar nelas.

    Armazenamento temporário (sem backend).
